# Ideia Central
A ideia do projeto é receber anotações (problemas) feitas por usuários.

# Formato da Anotação

- Título
- Descrição
- Visibilidade
- Prioridade
- Disciplinas (N <-> N)
- Anexos (1 <-> N)
- User (1 <-> 1)

# Formato da Disciplina

- Nome
- Sigla

# Formato do Anexo

- Nome do arquivo
- Vetores (JSON)
- URL Pública (Acesso ao arquivo)
